## Во всех заданиях использовать цикл FOR.
# Задание 2
Рассматривается множество целых чисел, принадлежащих отрезку [1125; 7856], которые делятся а 3 и не делятся на 7 и 5. Найдите количество таких чисел.
Для самопроверки: программа должна вывести число 1538.
```
count = 0
for x in range(1125, 7857):
    if x % 3 == 0 and x % 7 != 0 and x % 5 != 0:
        count += 1
print(count)
```
# Задание 3
Рассматривается множество целых чисел, принадлежащих отрезку [100; 999], которые являются палиндромом (одинаково читаются как слева направо, так и справа налево). Найдите сумму таких чисел.
Для самопроверки: программа должна вывести число 49500.
```
total = 0
for x in range(100, 1000):
    if str(x) == str(x)[::-1]:
        total += x
print(total)
```
# Задание 4
В программу вводится два натуральных числа: N и K. Выведите на экран последовательность, состоящую из N чисел с разностью K (последовательность начинается с единицы).
Пример ввода:
5
3
Пример вывода:
1 4 7 10 13
```
N = int(input())
K = int(input())
for i in range(N):
    print(1 + i * K, end=' ')
print()
```
# Задание 5
В программу вводится одно натуральное число N. Выведите на экран все значения функции f(x) = x*x + 5x - 7 в целых точках на отрезке [0; N].
Пример ввода:
10
Пример вывода:
-7 -1 7 17 29 43 59 77 97 119 143
```
N = int(input())
for x in range(N + 1):
    print(x * x + 5 * x - 7, end=' ')
print()
```
# Задание 6
В программу вводится натуральное число N, а затем N натуральных чисел. Требуется найти сумму чисел, которые делятся на 7 и являются трехзначными. В результате программа должна вывести одно число: сумму требуемых чисел.
Пример ввода:
5
15
749
222
17
651
Пример вывода:
1400
```
N = int(input())
total = 0
for _ in range(N):
    num = int(input())
    if 100 <= num <= 999 and num % 7 == 0:
        total += num
print(total)
```
# Задание 7
 В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000. Программа должна найти и вывести на экран максимальное число, кратное 7 и не кратное 9.
```
N = int(input())
max_num = -1
for _ in range(N):
    num = int(input())
    if num % 7 == 0 and num % 9 != 0:
        if num > max_num:
            max_num = num
print(max_num)
```
# Задание 8
В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000. Программа должна найти и вывести на экран минимальное двузначное число, восьмеричная запись которого оканчивается на 3.
```
N = int(input())
min_num = 1000
for _ in range(N):
    num = int(input())
    if 10 <= num <= 99:
        if oct(num)[-1] == '3':
            if num < min_num:
                min_num = num
print(min_num)
```
# Задание 9
В программу вводится число N. Программа должна вывести два столбца чисел. В первом столбце – числа от 1 до N, во втором – числа от N до 1.
Пример:
5
1 5
2 4
3 3
4 2
5 1
```
N = int(input())
for i in range(1, N + 1):
print(i, N - i + 1)
```
# Задание 10
В программу вводится число N. Требуется определить, является ли это число простым.
```
N = int(input())
is_prime = True
if N < 2:
    is_prime = False
else:
    for i in range(2, int(N ** 0.5) + 1):
        if N % i == 0:
            is_prime = False
            break
print("Yes" if is_prime else "No")
```
# Задание 11
В программу вводится натуральное число N. Требуется вывести на экран ряд чисел 1, 0.5, 0.25, 0.125, 0.0625, …, который содержит N элементов.
```
N = int(input())
value = 1
for _ in range(N):
    print(value, end=' ')
    value /= 2
print()
```
